# Digital Forensics Portfolio

Hands-on digital forensics labs and case studies, personal portfolio built while working through the [digital-forensics-lab](https://github.com/frankwxu/digital-forensics-lab) course by Dr. Frank Xu.

> Note: Section headers and technical terminology are in English (standard in digital forensics), while detailed analysis is written in French.
>
> Voir [CREDITS.md](CREDITS.md) pour l'attribution complète du cours source. Ce dépôt documente uniquement mon travail personnel, pas les supports pédagogiques originaux.

## About

Étudiant en cybersécurité, ce portfolio rassemble mes analyses pratiques réalisées cas par cas : registre Windows, réseau, mémoire vive, mobile/IoT. Chaque cas est documenté dans son propre dossier avec méthodologie, outils utilisés et résultats.
